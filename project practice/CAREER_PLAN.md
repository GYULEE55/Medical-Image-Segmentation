## 왜 의료 AI?

**핵심 논리**: 의공학 배경이 유일한 차별점.

```
같은 실력일 때:

일반 AI 회사 → "국비 6개월 + 프로젝트 3개" (경쟁자 수천명)
의료 AI 회사 → "의공학 졸업 + 의료 데이터 경험 + AI" (경쟁자 수십명)
```

- 의공학 졸업 → 의료 도메인 이해 (DICOM, 임상 변수, 환자 데이터)
- SEER 데이터 분석 경험 → 실제 의료 데이터 다뤄봄
- 졸업논문(EMG 센서) → 의료기기 + 데이터 수집 경험
- 첫 직장 잡고 1~2년 경력 쌓으면 다른 분야 이직 가능

### 프로젝트1: FaceGuard (완료)
- **태스크**: Classification + XAI
- **성과**: Macro-F1 43%→82% (1.9배), 최우수상
- **기술**: PyTorch, FSFM+ArcFace, RetinaFace, Streamlit
- **면접 포인트**: V1→V3 단계적 개선 스토리, 데이터 설계의 중요성

- **면접 포인트**: 의료 데이터 + 엔지니어링 스킬 (배포까지)
