처음 보는 분을 위한 10초 가이드

1) 샘플 선택/업로드 → 2) 병변 검출 결과 확인 → 3) 쉬운 한국어 해석 확인 → 4) 문헌 근거 확인. 근거가 부족하면 자동으로 제한해 오해 가능성을 낮춥니다.
