## 2. 코테 방향 (백준 4단계 벽)

결론: **외우기보다 복습 루프**가 정답.

### 현재 단계에서 해야 할 것
- 1차: 브론즈/실버 하위에서 "구현+배열" 10문제만 반복
- 2차: 같은 유형을 24시간 뒤 다시 풀기 (기억 고정)
- 3차: 실패 문제는 "오답노트" 3줄 템플릿 작성

오답노트 템플릿:
1) 내가 놓친 조건
2) 정답 핵심 아이디어 1줄
3) 다음에 볼 체크포인트

### 주간 목표 (코테)
- 평일 1문제 + 복습 1문제 (총 30~40분)
- 주말 2시간: 한 주 오답만 재풀이

**면접 포인트**:
> "무작정 문제 수를 늘리기보다, 실패 패턴을 구조화해 재발을 줄이는 학습 루프를 적용했습니다."
